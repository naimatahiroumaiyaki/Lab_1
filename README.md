Lab 1 Part A and B 

This lab focuses on implementing and comparing search algorithms for drone path planning. Part A covers uninformed search (BFS, DFS, DLS, and IDS).
Part B covers informed search (UCS, Greedy Best-First Search, A*, and Weighted A*) using heuristics and weighted terrain costs. The project includes experiments, custom maps, and analysis of algorithm performance.
